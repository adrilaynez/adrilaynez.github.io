Before we had ChatGPT, we had the Perceptron.

The First Neuron#

In 1958, Frank Rosenblatt created the Perceptron. It wasn’t code; it was hardware. Cables connected potentiometers representing weights.

The Mark I Perceptron machine.

It failed to solve XOR, leading to the “AI Winter”. But the math survived.
